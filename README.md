# Animatronic-Eyes-Project
This repository contains the final deliverables for the Animatronic Eyes Project, a term project for the 2103532 CAD & Digital Workflow course.

The repository includes:

* Fusion360 CAD file (.f3z), including all designed parts and commercial components.
* Multiple realistic rendered images of final design
* A full assembly instruction animation
* Static structural simulation results
* STL files for 3D printing.
* Documentation of the final build, including photos and performance videos
